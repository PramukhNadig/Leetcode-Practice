For this problem what is necessary is a Stack since you need to check the most recently used open paren.

When an open paren is seen, push it to the stack, else pop the stack and compare and ensure the variables make sense.

Finally, after the string is iterated through, if the stack is not empty then the parentheses are not valid, else they are.



```Java
class Solution {
    public boolean isValid(String s) {
        Stack<Character> stack = new Stack<Character>();
        
        for(int i = 0; i < s.length(); i++){
            char c = s.charAt(i);
            if(c == '(' || c == '{' || c == '['){
                stack.push(c);
            }else{
                
                if(stack.isEmpty()){
                    return false;
                }
                
                char p = stack.pop();
                if((p == '(' && c != ')') || (p == '[' && c != ']') ||  (p == '{' && c != '}')){
                    return false;
                }
            }
        }
        
        if(stack.isEmpty()){
            return true;
        }else{
            return false;
        }
    }
}
```
